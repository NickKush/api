# 0.1.0 (2016-XX-XX)
- **chat**
    - [`/chat/message`](chat.md#Новое-сообщение) добавлено поле с активными бонусами отправителя сообщения
- **common**
    - [`/api/user`](common.md#Данные-пользователя), [`/api/user/current`](common.md#Данные-текущего-пользователя), [`/api/user/full`](common.md#Полные-данные-пользователя) добавлено поле `slug` с коротким именем используемым для формирования ссылок
    - добавлена [роль](common.md#Данные-текущего-пользователя) администратора стримов `streamAdmin`
    - [`/api/stream`](common.md#Данные-стрима), [`/api/content`](common.md#Список-элементов-контента) добавлено поле онлайн статуса стрима `active`
- **poll**
    - [`poll/update`](poll.md#Обновлённые-данные) добавлено уведомление о обновлении активного голосования
- **smile**
    - [`/api/smile`](smile.md#Доступные-смайлы) добавлен флаг анивированных смайлов `animated`
- **store**
    - добавлены запросы и уведомления магазина [`/api/store/*`](store.md)

#### BREAKING CHANGES
- **chat**
    - убраны неработающие оповещения клиентов `/chat/user/join` и `/chat/user/leave`
- **common**
    - [`/api/stream`](common.md#Данные-стрима) изменён состав полей и их содержимое
        - запрос больше не принимает `id` или `owner`, только `slug` стримера (можно получить из ответа [`/api/user`](common.md#Данные-пользователя))
        - убран параметр запроса `options`, список плееров возвращает на каждый запрос
        - убрано поле ответа `slug`, используйте `owner.slug` с тем же значением
        - `category` всегда null (до появления нового функционала категорий)
        - `thumbnail` содержит абсолютную ссылку на превью онлайн стрима
        - `active` отвечает за статус активности стрима, а не онлайна, если стрим неактивен то он недоступен никому кроме администраторов стримов
        - изменены поля плееров, iframe плеера формируется вручную по полям `provider` и `channel`, поле `code` с HTML плеера убрано 
        - добавлены поля: `online`
        - убраны поля: `image`, `adult`, `hidden`
    - убран неработающий запрос на видео превью стрима `/api/stream/preview`
    - изменения в [полях элемента контента](common.md#Поля-элемента-контента): поля приведены в соответствия с изменениями в данных стрима
    - убраны запросы работы с расписанием, майлстоунами, сезонами соревнования, уровнями и аукционом в связи с неактуальностью. Могут вернуться в дальнейшем в том же или обновлённом виде
    - убраны запросы мастерстримеров `/api/masterstreamer/list` и `/api/masterstreamer/icon/list`, используйте [список смайлов](smile.md#Доступные-смайлы) и [иконок](smile.md#Список-иконок) с фильтрацией по полю `user` для смайлов и иконок конкретных стримеров. Мастерстримеры теперь упоминаются как стримеры с партнёркой
- **smile**
    - [`/api/smile`](smile.md#Доступные-смайлы) убраны поля `level`, `masterStreamerLevel`, `siteLevel` (их возможное наличие стоит игнорировать)
