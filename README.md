API:  
------
###### Уровень доступа:
  - **`P`** - Публичный, авторизация не обязательна
  - **`A`** - Все авторизованные пользователи
  - **`M`** - Авторизованные модераторы
  - **`C`** - Закрытый, для внутреннего использования
